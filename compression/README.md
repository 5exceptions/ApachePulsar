# ApachePulsar Producer & Consumer (Compression)


Pulsar currently supports the following types of compression:
  -   LZ4
  -   ZLIB
  -   ZSTD
  -   SNAPPY
    

[Reference Link](https://pulsar.apache.org/docs/concepts-messaging/#compression)
